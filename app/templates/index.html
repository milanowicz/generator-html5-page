Bower Installation 

$ npm install -g bower

Bower Usage

Using the dependencies listed in the current directory's bower.json

$ bower install

Using a local or remote package

$ bower install

Using a specific version of a package

$ bower install #

Install package and safe into bower.json

$ bower install  -S

GruntJS Installation

Install before your use GruntJS the first time! 

$ npm install -g grunt-cli

Install all Grunt modules in your project directory and let's roll 

$ cd project/
$ npm install

Development and Release

Generate development files 

$ grunt dev

Let Grunt watch if some file are change and let him do his task 

$ grunt watch

Do run all Tasks for a Release 

$ grunt

Special Tasks

Create a archiv from the Public and Documentation 

$ grunt zip

Git Configuration

set a config value globally for this user 

$ git config --global user.name "My Name"

set a config value globally for the user email 

$ git config --global user.email "a@b.c"

Fork a Project 

$ git clone url
$ cd project
$ (edit files)
$ git add (files)
$ git commit -m 'Explain what I changed'
$ git pull origin master
$ git push origin master

Branch and Merge

create a branch and immediately switch to it 

$ git checkout -b [branch]
$ (edit files)
$ git add (files)
$ git commit -m 'Explain what I changed'
$ git checkout master
$ git merge [branch]
$ git pull origin master
$ git push origin master
